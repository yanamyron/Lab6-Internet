Прізвище:

1.Bootstrap - це відкрите CSS-фреймворк, розроблений командою Twitter. Він містить набір заздалегідь написаних стилів, компонентів та JavaScript-розширень, які допомагають розробникам швидше створювати сучасні та адаптивні веб-сайти.

2.У чистому JavaScript немає функції для доступу до елементу за іменем без використання інших атрибутів. Однак, використовуючи jQuery, можнавикористовувати селектори CSS, наприклад, $('[name="ім\'я"]').

3.Для доступу до елементу за класом використовується функція getElementsByClassName у JavaScript.

4.У термінах DOM, document представляє цілу HTML-сторінку, що відкрита в браузері. Він є основною точкою входу для взаємодії з DOM.

5.DOM (Document Object Model) - це програмований інтерфейс для роботи з документами HTML і XML. Він представляє структуру документа у вигляді дерева об'єктів, де кожен вузол представляє частину документа, таку як елемент, атрибут, текстовий вузол тощо. DOM надає засоби для динамічного змінення вмісту, структури та стилів сторінки під час її виконання.
